# Agent Instructions

## Python Environment
When working on this project, always use the virtual environment located in the project root at `venv`.
- To run Python scripts, use `.\venv\Scripts\python.exe`
- To install packages, use `.\venv\Scripts\pip.exe`
- The project requires Python 3.11 or 3.12 for CUDA PyTorch compatibility. Do not use Python 3.14.
